Procurement Process Automation

A Google Sheets-based system that cut data entry errors by over 50% and boosted efficiency.

The Business Problem

As a growing MSME, the company's procurement process was entirely manual. Purchase orders were created and tracked in basic spreadsheets, a method that was time-consuming, prone to costly data entry errors, and lacked a centralized, auditable trail. To support the company's expansion, I was tasked with developing an efficient and scalable automated PO system.

The Solution: An Automated Workflow

I designed and implemented a comprehensive PO automation tool using Google Sheets and Google Apps Script. The solution transformed the procurement workflow into three streamlined stages:

  1. Smart PO Generation: A user selects a supplier from a dropdown, and the system automatically populates the PO template with the correct addresses, GST number, and contact details from a central database, eliminating manual entry.
  2. One-Click PDF Export & Logging: A custom menu button exports the completed PO as a formatted PDF to a shared Google Drive folder. Simultaneously, it records all key details (PO number, date, supplier, etc.) in a central log sheet, creating an instant audit trail.
  3. Integrated Email Distribution: A user can input recipient emails and, with a simple dropdown selection, the system automatically emails the PO PDF to the specified contacts, logging the "sent" status and time.

Technical Implementation

// Paste your Google Apps Script for PO number generation here
function generatePoNumber() {
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName("PO_Sheet");
  const date = new Date();
  const branchCode = "MDU";
  const year = date.getFullYear().toString().slice(-2);
  const month = ("0" + (date.getMonth() + 1)).slice(-2);
  const day = ("0" + date.getDate()).slice(-2);
  const randomNum = Math.floor(100 + Math.random() * 900);
  
  const poNumber = `PO-${branchCode}-${year}${month}${day}-${randomNum}`;
  sheet.getRange("C5").setValue(poNumber); 
}

Outcome & Impact

  • Reduced Errors by over 50%: By automating supplier data entry, the system drastically cut down on costly manual errors.
  • Improved Operational Efficiency: The one-click PDF generation and emailing saved significant administrative time, allowing the accounts manager to focus on higher-value tasks.
  • Established an Audit Trail: The automated logging system created a centralized, digital, and auditable trail for all purchase orders, improving financial governance.
